Olof's me-page in the course ramverk2.

## Install

1. Clone the repo
2. `cd` into the repo
3. `npm install`
4. `npm start`

The server listens to the port set in `DBWEBB_PORT` or defaults to `3000`.

## Test
Run `npm test` to run linters and upcoming tests.
